# World Dictionary

**Status:** Living editorial vocabulary for the AlephBits World Engine.  
**Authority file:** [dictionary.yaml](dictionary.yaml)  
**App counterpart:** in-app World Dictionary (aggregates from compiled `lesson.json`).

## Rules

1. **Dictionary-first** — search this file before inventing a new id.  
2. **Reuse** existing ids and aliases (`flashlight` not `flash_light`).  
3. **New ids** start as `status: proposed`. Never auto-approve.  
4. **Categories** are fixed for now: `objects`, `creatures`, `plants`, `symbols`, `places`.  
5. Prefer sparse, characteristic elements (3–5 per book).
